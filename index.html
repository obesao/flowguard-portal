Funções de Detecção

Liga/desliga cada tipo de ataque detectado individualmente. Desabilitar não fecha ataques já ativos, só para de abrir/atualizar novos a partir de agora.

Carregando...

Mitigação

Estratégia sugerida (botão "Aplicar Sugestão" na aba Ataques) por tipo de ataque — RTBH bloqueia o prefixo inteiro via BGP; Descartar (FlowSpec) derruba só o tráfego que casa o padrão do ataque; Limitar banda (FlowSpec) não derruba nada, só limita a velocidade desse tráfego. O botão "Mitigar" continua sempre RTBH, como ação manual de emergência.
